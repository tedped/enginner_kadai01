# ① 課題名

エレベーターシミュレーション

## ② 課題内容（どんな作品か）

- エレベーターの動作をシミュレーションする作品です。階数ボタンを押すと、ボタンに紐づくエレベーターが昇降します。
- エレベーターシャフトの基数及び建物の階数を自由に増減できます。

- PW: なし

## ⑤ 工夫した点・こだわった点

- JS の「状態管理」「非同期処理」を勉強しながら取り組みました。

## ⑥ 難しかった点・次回トライしたいこと（又は機能）

- エレベーター内の階数ボタンを実装しましたが、エレベーター外の矢印ボタンを実装できていません。
- 複数の実行リクエストを束ねるには、「ブラウザ描写」「実行」「制御」の役割ごとにファイルを分けた方が良いらしいですが、提出期限までに間に合いませんでした。
- PHP でエレベーターの利用状況をデータ管理して、データを参照することでより効率的な制御となるような仕組みを取り入れたいと思っていました。
